# Yellalena Tap

## How do I install these formulae?

`brew install yellalena/tap/<formula>`

Or `brew tap yellalena/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "yellalena/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
